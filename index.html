Wildiz

Wild Fire Event Visualization Tracker

WildViz is a web map application that visualizes wildfire events. WildViz is inspired by wildfire outbreaks occurring on our planet earth.

All data visualized on WildViz is obtained from the EONET (Earth Observatory Natural Event Tracker) repository using a customized Django API.

If you are interested in knowing about the EONET, kindly click here

WildViz is for visualization and general information purposes only and should not be construed as "official" with regard to a spatial or temporal extent.
